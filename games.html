GAMES PROJECTS


A Collection of Games Made By Me:

Below are a few games that I have made through RPG Maker MV for some classes at Temple University.

  • GAMES CLASS PROJECT – ZOOM DOOM

    Zoom Doom is a project made by myself and two other students. The main idea was derived from the group leader. However, the game's overall look, design, and mechanics were made by me. Essentially, one student was the music guy and the other was the ideas guy. I was the one that had the technical knowledge and know-how to make everything work.

    SOFTWARE: RPG MAKER MV
    TIME: 4 DAYS

  • COMPOSITION CLASS PROJECT – TALES OF FATE

    Tales of Fate is an isekai game made entirely by me for a class prject in Temple University.The other students in the group acting more like beta testers for the game. One student helped with the story line and the other made a little website for the game. I'm not quite sure if the website is still active. Either way, 100% of the work on the game for this project was done by myself.

    SOFTWARE: RPG MAKER MV
    TIME: 1 WEEK
